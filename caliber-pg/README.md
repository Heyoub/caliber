CALIBER PostgreSQL Extension
============================

Notes on packaging
------------------
- The extension SQL that creates `#[pg_extern]` functions is generated by
  `cargo pgrx package` / `cargo pgrx install`.
- Do not ship the repo's static `sql/caliber_pg--<version>.sql` as the
  install script. It contains DDL only and will omit functions.
- The `make package` target builds the pgrx package and copies the generated
  SQL into the PGXN zip.

Local install (PG18)
--------------------
1) Build/install:
   cargo pgrx install --package caliber-pg --pg-config "/usr/lib/postgresql/18/bin/pg_config"
2) Enable:
   sudo -u postgres psql -d caliber -c "CREATE EXTENSION caliber_pg;"
