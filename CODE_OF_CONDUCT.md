# Code of Conduct

## Our Pledge

We're here to build cool shit and help each other learn. Be respectful, be constructive, and be professional.

## Our Standards

**Do:**
- Be respectful and considerate
- Give and accept constructive feedback
- Focus on what's best for the project
- Show empathy towards others
- Assume good intentions

**Don't:**
- Harass, insult, or attack others
- Share others' private information
- Troll, spam, or derail discussions
- Make unwelcome sexual advances
- Act unprofessionally

## Scope

This applies to:
- GitHub issues, PRs, and discussions
- Project communication channels
- Representing the project publicly
- Project events and spaces

## Enforcement

Violations can be reported to the project maintainers. All reports will be reviewed and investigated.

Consequences may include:
- Warning
- Temporary ban
- Permanent ban

## Attribution

Inspired by the [Contributor Covenant](https://www.contributor-covenant.org/), but shorter and more direct.

---

**TL;DR:** Don't be a jerk. We're all here to build something useful.
